# node-esm-example
Example JavaScript console app using esm for import/export.

## use

Clone the repo, then:
```
cd node-esm-example
npm install
npm start
```

The can be run using `node index.js` or `npm start`.

## generation
The application was generated using npm init:
```
mkdir esm-test && cd esm-test
npm init esm --yes
```
